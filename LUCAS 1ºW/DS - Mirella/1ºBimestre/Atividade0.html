Desenvolvimento de Sites - HTML

HTML:

A sigla HTML significa Hyde Linguagem, que em
português significa Linguagem de Marcação de
Hiper Texto. É uma linguagem de marcação
utilizada para páginas da Web ou aplicativos.
Sendo desenvolvido na década de 90 por Tim
Berner-Lee.

CSS:

CSS é a sigla para Cscading Style Sheets, ou
seja, Folhas de Estilo em Cascatas. É uma
maneira de dar estilo ao código criado por
linguagens como HTML, XML ou XHTML, por
exemplo. De forma prática, ela funciona como
uma camada de pesonalização ao conteúdo
visível.
